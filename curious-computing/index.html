Label Maker for Curious Computing

Just click below to edit fields, then print. This bit won't print. Here's an example

This is not a clever form, there is no submit button - once you close the browser or reload, it is gone!

The point is to allow you to make a label that you can print out for the event


The main body text is formatted using markdown . (This means you can use a real editor to write things, then paste it in) Click on a field to edit it. Please also over-write the top heading (CPU Name/Architecture), removing the ARM-logo for non-ARM stuff (Bonus points if you find a logo image for whatever architecture the product is based on!). Here is an example label for a Psion Series 5 you might just prefer to edit that...
There are four 'items' to edit

  • Make/Model
  • Date
  • Details (Body)
  • Link for further information

Please make one label for each piece of hardware you have, if possible. Refreshing this page will allow (/force) you to start again
